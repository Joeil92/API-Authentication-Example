# API REST

## Description

API REST construite avec Node.js et Express.

## Installation

1. Clonez le dépôt :

   ```
   git clone https://github.com/Joeil92/Athleticsperformances-back.git
   cd nom-du-repo
   ```

2. Installer les dépendances

    ```
    npm install
    ```

3. Run docker

    ```
    docker compose up -d
    ```

4. Run le serveur de dev

    ```
    npm run dev
    ```
